# Sending Mail Summaries

This is an application that looks for customers with a debit balance.
Export the account summary to an excel file and then send it by email.

## Environment variables

    - URBANO_HOST
    - URBANO_DB
    - URBANO_PASS
    - URBANO_USER
    - MAIL_HOST
    - MAIL_USER
    - MAIL_PASSWORD
    - MAIL_FROM
    - MAIL_INFO
    - MAIL_CC
    - MAIL_TESTING
    - MAIL_LIMIT
    - FILE_PATH_PREFIX
    - INDEX_DEBTOR
    - TESTING
